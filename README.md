Тема: Разработка веб-приложения «Личный кабинет волонтёра» с системой мероприятий, рейтингов и уведомлений



Приложение предназначено для автоматизации работы волонтёрского центра.  
Оно позволяет волонтёрам:

- регистрироваться и входить в систему (JWT-аутентификация);
- просматривать актуальные мероприятия и записываться на них;
- читать ленту новостей (посты координатора);
- оставлять комментарии и ставить лайки под постами;
- получать уведомления о событиях;
- накапливать рейтинг за активность (баллы за комментарии, лайки, участие в мероприятиях);
- управлять своим профилем.

Координатор (администратор) имеет доступ к панели управления, где может:

- создавать, редактировать и удалять мероприятия;
- управлять заявками волонтёров;
- публиковать и удалять посты в ленте;
- просматривать список волонтёров и их рейтинг.


!! Стек технологий

! Бэкенд
- Python 3.12+
- Django 6.0
- Django REST Framework (DRF)
- djoser + SimpleJWT (аутентификация)
- PostgreSQL (база данных)
- Poetry (управление зависимостями)

! Фронтенд
- React 18
- TypeScript
- Vite (сборка)
- Tailwind CSS (стили)
- Zustand (управление состоянием)
- React Router DOM (маршрутизация)
- Axios (HTTP-запросы)
- Lucide React (иконки)

🚀 Запуск проекта
1. Клонирование по ssh
2. cd volunteer_portal
3. cd backend
4. pip install poetry - если нету
5. poetry install

6. Создать файл .env
7. Добавить
  SECRET_KEY=django-insecure-сгенерируй-свой-ключ
  DEBUG=True
  DB_NAME=volunteer_db
  DB_USER=postgres
  DB_PASSWORD=твой_пароль
  DB_HOST=localhost
  DB_PORT=5432

8. poetry run python manage.py migrate
9. poetry run python manage.py runserver
10. cd frontend
11. npm install
12. npm run dev

Суперюзер не нужен т.к на данный момент админка доступна всем зарегестрировавшемся пользователям http://127.0.0.1:8000/admin

POST	/auth/users/	регистрация
POST	/auth/jwt/create/	вход (JWT)
GET	/api/events/	список мероприятий
POST	/api/applications/	запись на мероприятие
GET	/api/posts/	лента постов
POST	/api/likes/	лайк поста
GET	/api/notifications/	уведомления
GET	/api/ratings/me/	рейтинг текущего пользователя
GET	/api/admin/users/	список волонтёров (только staff)
